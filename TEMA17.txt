1. ¿Cuál es el paso inicial para ejecutar una sentencia SELECT en una base de datos relacional?
   A) Crear un objeto de tipo ResultSet
   B) Utilizar el método executeQuery
   C) Conectar con la base de datos
   D) Crear un objeto de la clase Statement
   Respuesta correcta: D) Crear un objeto de la clase Statement

2. ¿Qué método se utiliza para obtener el resultado de una consulta SELECT en Java?
   A) getResult()
   B) fetchResult()
   C) executeQuery()
   D) fetchQuery()
   Respuesta correcta: C) executeQuery()

3. ¿Qué tipo de excepción puede lanzar el método executeQuery al ejecutar una consulta en Java?
   A) SQLException
   B) NullPointerException
   C) IOException
   D) ClassNotFoundException
   Respuesta correcta: A) SQLException

4. ¿Qué se utiliza para recorrer y obtener los valores de un objeto ResultSet en Java?
   A) Método fetchNext()
   B) Método next()
   C) Método getNext()
   D) Método moveNext()
   Respuesta correcta: B) Método next()

5. ¿Qué tipo de comillas se utilizan para valores de tipo cadena en una consulta SELECT en una base de datos relacional?
   A) Comillas simples ('')
   B) Comillas dobles ("")
   C) Comillas angulares (<>)
   D) Comillas invertidas (``)
   Respuesta correcta: A) Comillas simples ('')

6. ¿Qué programa de software libre contiene un sistema gestor de bases relacionales con MySQL?
   A) Xcode
   B) XAMPP
   C) Eclipse
   D) Visual Studio
   Respuesta correcta: B) XAMPP

7. ¿Qué ventaja ofrecen las bases de datos relacionales para el tratamiento de grandes cantidades de información? 

A) Almacenamiento en ficheros secuenciales. B) Acceso lento a la información. 
C) Optimización para almacenar grandes volúmenes de información. 
D) Limitación en la cantidad de datos almacenados. 

Respuesta correcta: C) 

8. ¿Qué se recomienda en programación para evitar la repetición excesiva de código al conectar a una base de datos? 
A) Repetir el código tantas veces como sea necesario. 
B) Reutilizar el código escrito lo mínimo posible. 
C) No preocuparse por la estructura del programa. 
D) Utilizar diferentes códigos para cada operación. 

Respuesta correcta: B) 

9. ¿Por qué es importante elegir adecuadamente el tipo de base de datos a utilizar en un proyecto de programación? 

A) No es relevante, ya que todas las bases de datos son iguales. 
B) Para complicar el desarrollo del proyecto. 
C) Para garantizar un almacenamiento eficiente de la información. 
D) Para aumentar la cantidad de código necesario. 

Respuesta correcta: C) 

10. ¿Qué tipo de operaciones básicas se pueden realizar en una base de datos relacional? 
A) Solo inserción de datos. 
B) Inserción, consulta, borrado y modificación de datos. 
C) Solo consulta de datos. 
D) Solo modificación de datos. 

Respuesta correcta: B)

11. ¿Qué papel juega la programación orientada a objetos en la gestión de bases de datos? 
A) No tiene relación con la gestión de bases de datos. 
B) Facilita la reutilización de código y la estructuración de los programas. 
C) Complica la conexión a la base de datos. D) Limita las operaciones que se pueden realizar en la base de datos.

12. ¿Qué ventaja ofrece una interfaz gráfica en la gestión de la información de una base de datos?
