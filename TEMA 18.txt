¿Cuál es una característica clave de las bases de datos orientadas a objetos? 
A) Utilizan SQL como lenguaje de consulta.
B) Almacenan objetos en lugar de filas y columnas. 
C) No permiten la herencia entre clases. 

Respuesta correcta: B) Almacenan objetos en lugar de filas y columnas.

¿Qué lenguaje se utiliza para realizar consultas en bases de datos orientadas a objetos? 
A) SQL 
B) QBE 
C) Java 

Respuesta correcta: B) QBE

¿Qué gestores de bases de datos soportan las bases de datos orientadas a objetos? 
A) Oracle y Microsoft SQL Server 
B) MySQL y PostgreSQL 
C) MongoDB y Redis 

Respuesta correcta: A) Oracle y Microsoft SQL Server

¿Qué tipo de relación se puede establecer entre objetos en bases de datos orientadas a objetos? 
A) Claves primarias y foráneas 
B) Herencia y composición 
C) Filas y columnas 

Respuesta correcta: B) Herencia y composición

¿Cómo se actualizan los objetos en una base de datos orientada a objetos? 
A) Con la sentencia UPDATE de SQL 
B) Modificando el objeto y volviéndolo a insertar en la base de datos 
C) Eliminando y volviendo a crear el objeto 

Respuesta correcta: B) Modificando el objeto y volviéndolo a insertar en la base de datos
