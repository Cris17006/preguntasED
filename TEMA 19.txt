1. ¿Cuál es el propósito principal de los sistemas de control de versiones en el desarrollo de software?

	a)Aumentar la complejidad del desarrollo.
	b)Sincronizar fácilmente el código entre equipos de desarrollo.
	c) Reducir la productividad del equipo.
	d) Restringir el acceso al código fuente.

	Opción correcta: b


2. ¿Cuáles son las diferencias clave entre los repositorios centralizados y distribuidos en los sistemas de control de versiones?

	a) Los repositorios centralizados permiten a cada usuario tener una copia local del 	repositorio.
	b)Los repositorios centralizados permiten a cada usuario de acceso controlado por un 	conjunto de usuarios.
	c) Los repositorios centralizados requieren conexión constante a internet.
	d) Los repositorios distribuidos permiten trabajar sin conexión a internet.

	Opción correcta: d.

3.¿Qué ventajas ofrece Git como sistema de control de versiones en comparación con SVN?

	a)Mayor simplicidad en la gestión de ramas
	b) Facilidad para realizar bloqueo de archivos.
	c) Mejor seguimiento de modificaciones realizadas.
	d) Mayor compatibilidad con sistemas operativos antiguos

	Opción correcta: a.

4. ¿Cuáles son los comandos básicos utilizados en Git para gestionar cambios en un repositorio local y remoto?

	a) pull y checkout
	b) add y commit
	c) clone y log
	d) push e init

	Opción correcta: b.
