1. ¿Qué plataforma es conocida por permitir el control de versiones con múltiples herramientas como Git, SVN, etc.?
   A) GitLab
   B) GitHub
   C) GNU Savannah
   D) Gitea

   Respuesta correcta: C) GNU Savannah

2. ¿Qué característica diferencia a GitLab de GitHub en cuanto a la granularidad de los permisos de compartición de código?
   A) GitLab permite establecer permisos a nivel de repositorio y archivos
   B) GitHub ofrece mayor detalle en la fijación de permisos
   C) GitLab permite determinar qué parte de un repositorio compartir con colaboradores
   D) GitHub no permite compartir código con colaboradores

   Respuesta correcta: C) GitLab permite determinar qué parte de un repositorio compartir con colaboradores

3. ¿Qué alternativa se plantea en el PDF para realizar copias de seguridad del código fuente de una empresa?

   A) Utilizar un servidor local para almacenar las copias de seguridad
   B) Contratar un servicio de copias de seguridad externo
   C) Utilizar servicios basados en la nube para realizar copias de seguridad
   D) No realizar copias de seguridad
   Respuesta correcta: C) Utilizar servicios basados en la nube para realizar copias de seguridad

4. ¿Qué tipo de arquitectura Cloud permite utilizar el software a través de aplicaciones con conexiones remotas?

   A) IaaS (Infraestructura como Servicio)
   B) PaaS (Plataforma como Servicio)
   C) SaaS (Software como Servicio)
   D) NaaS (Network as a Service)
   Respuesta correcta: C) SaaS (Software como Servicio)

5. ¿Qué herramienta de escritorio se menciona en el PDF para gestionar proyectos en GitHub?

   A) GitKraken
   B) GitHub Desktop
   C) GitLab Desktop
   D) GitHub Manager
   Respuesta correcta: B) GitHub Desktop
